Welcome to PerfWatcher

PerfWatcher is a web-based frontend for RRD datas collected with Collectd.

You can use Perfwatcher with Collectd >= 5.0 or use Collectd-pw, our own flavor of Collectd.

Features

  • Host arrangement in folders and sub-folder
  • RRD graph panning and zooming
  • RRD aggregation
  • jQuery based web interface
  • PHP / JSON backend
  • Unix top displaying
  • Server status
