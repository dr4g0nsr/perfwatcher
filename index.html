Welcome to PerfWatcher

PerfWatcher is a web-based frontend for RRD datas collected with Collectd.

Features

  • Host arrangement in folders and sub-folder
  • RRD graph panning and zooming
  • RRD aggregation
  • jQuery based web interface
  • PHP / JSON backend
  • Unix top displaying
  • Server status

Screenshots

Folder view where you can, for example, view total cpu used for all server behind it. Host view. Custom view to display graph form different hosts on the same page. Top functionnality to see process list in the past
