Frequently Asked Questions

  • Q: Can Perfwatcher working with the original version of Collectd ?
  • R: Yes, but lot of feature will not work, particularly for the server part.
  • Q: What is the difference between Collectd and Collectd-pw
  • R: Collectd-pw is the latest version of Collectd (currently the 5.1) with a set of patch to improve Collectd or adding some features nedeed by Perfwatcher.

  • Q: How many servers can be monitored with Collectd / PerfWatcher ?
  • R: The limit is your server. Collectd is hightly scallable and can be load-balanced. Actually I have 5600 servers monitored by 1 Collectd server at work.
