Agent installation

Until Collectd-pw has no package available, you need to compile it as explain bellow.
You can also use standard Collectd, but some amazing features as Top process will not work.

Prerequisites

To build Collectd-pw, you need some tools :

  • On RedHat like : 
    yum groupinstall 'Development tools'
  • On Debian like : 
    aptitude install build-essential

Getting sources

Two possibility :

  • Clone GitHub repo : 
    git clone -b 'release/5.1' https://github.com/perfwatcher/collectd-pw.git
  • Download last stable version : 
    wget https://github.com/downloads/perfwatcher/collectd-pw/collectd-5.1.0.20121017.tar.gz

Compile and install Collectd-pw

cd collectd
./configure --enable-top --enable-cpu
make -j
make install

Configure Collectd-pw

Below a sample of configuration for an agent : 

TypesDB     "/usr/share/collectd/types.db"
Interval     60

LoadPlugin cpu
LoadPlugin df
LoadPlugin disk
LoadPlugin interface
LoadPlugin load
LoadPlugin memory
LoadPlugin processes
LoadPlugin swap
LoadPlugin top
LoadPlugin uptime

LoadPlugin network
<Plugin network>
	# IP to be replaced by Collectd server's IP
	Server "192.168.0.73" "25826"
	MaxPacketSize 65535
</Plugin>
