Server installation

Until Collectd-pw has no package available, you need to compile it as explain bellow.
You can also use standard Collectd, but some amazing features as Top process will not work.

Prerequisites

To build Collectd-pw, you need some tools :

  • On RedHat like : 
    yum groupinstall 'Development tools'
yum install zlib-devel rrdtool-devel rdtool
  • On Debian like : 
    aptitude install build-essential
aptitude install zlib1g-dev librrd-dev rrdtool
And json-c >= 0.10. See install page : https://github.com/json-c/json-c/wiki until there is a json-c package >= 0.10 in your distribution.

Getting sources

To possibility :

  • Clone gitHub repo : 
    git clone -b 'release/5.1' https://github.com/perfwatcher/collectd-pw.git
  • Download last stable version : 
    wget https://github.com/downloads/perfwatcher/collectd-pw/collectd-5.1.0.20121017.tar.gz

Compile and install Collectd-pw

cd collectd
./configure --enable-top --enable-cpu --enable-rrdtool \
--enable-jsonrpc --enable-notify_file
make -j
make install
