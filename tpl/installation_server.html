Server installation

Until Collectd-pw has no package available, you need to compile it as explain bellow.
You can also use standard Collectd, but some amazing features as Top process will not work.

Prerequisites

To build Collectd-pw, you need some tools :

  • On RedHat like : 
    yum groupinstall 'Development tools'
yum install zlib-devel rrdtool-devel rrdtool
  • On Debian like : 
    aptitude install build-essential
aptitude install zlib1g-dev librrd-dev rrdtool

Compile json-c

wget https://github.com/downloads/json-c/json-c/json-c-0.10.tar.gz
tar xzf json-c-0.10.tar.gz
cd json-c-0.10
./configure --prefix=/usr
make -j
make install
cp json_object_iterator.h /usr/include/json # Bug in libjson install script
cp /usr/lib/libjson.* /usr/lib64/ # Needed on RHEL x86_64

Compile libmicrohttpd

wget http://ftp.gnu.org/gnu/libmicrohttpd/libmicrohttpd-0.9.22.tar.gz
tar xzf bmicrohttpd-0.9.22.tar.gz
cd libmicrohttpd-0.9.22
./configure --prefix=/usr
make -j
make install
cp /usr/lib/libmicrohttpd.so* /usr/lib64/ # Needed on RHEL x86_64

Getting sources

Two possibility :

  • Clone GitHub repo : 
    git clone -b 'release/5.1' https://github.com/perfwatcher/collectd-pw.git
  • Download last stable version : 
    wget https://github.com/downloads/perfwatcher/collectd-pw/collectd-5.1.0.20121025.tar.gz

Compile and install Collectd-pw

cd collectd
./configure --enable-top --enable-cpu --enable-rrdtool \
--enable-jsonrpc --enable-notify_file --enable-basic_aggregator \
--prefix=/usr
make -j
make install
cp src/types-perfwatcher.db /etc
  • On RedHat like : 
    cp contrib/redhat/init.d-collectd /etc/init.d/collectd
chmod a+x /etc/init.d/collectd
  • On Debian like : 
    curl -o /etc/init.d/collectd 'http://git.tokkee.org/?p=pkg-collectd.git;a=blob_plain;f=debian/collectd-core.collectd.init.d;hb=HEAD'
chmod a+x /etc/init.d/collectd

Configure Collectd-pw

Below a sample of configuration for a server :
/etc/collectd/collectd.conf on Debian like
/etc/collectd.conf on RedHat like 

TypesDB     "/usr/share/collectd/types.db"
TypesDB     "/etc/types-perfwatcher.db"
Interval     60

LoadPlugin cpu
LoadPlugin df
LoadPlugin disk
LoadPlugin interface
LoadPlugin load
LoadPlugin memory
LoadPlugin processes
LoadPlugin swap
LoadPlugin top
LoadPlugin uptime

LoadPlugin network
<Plugin network>
	Listen "0.0.0.0" "25826"
	MaxPacketSize 65535
</Plugin>

LoadPlugin rrdtool
<Plugin rrdtool>
	DataDir "/var/lib/collectd"
</Plugin>

LoadPlugin jsonrpc
<Plugin jsonrpc>
	Port "8080"
	MaxClients 64
</Plugin>

LoadPlugin notify_file
<Plugin notify_file>
	DataDir "/var/lib/collectd/_notification"
</Plugin>

LoadPlugin basic_aggregator
<Plugin basic_aggregator>
	Aggregators_config_file "/<path to perfwatcher>/etc/aggregator.conf"
</Plugin>
